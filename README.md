# imc2026
Ciência de dados para calcular IMC de um grupo de pessoas

Crie uma planilha com dados de 20 pessoas (NOME, IDADE, PESO, ALTURA)

E responda:
*qual a média de idade?
*qual o menor peso?
*qual a maior altura?
Criar uma coluna chamada IMC ("Índice de Massa Corporal")
Responder
*Qual o grupo IMC dominante nas 20 pessoas?

**Utilizar Pandas na codificação**

Colocar o código python no repositório, com o arquivo nomeado como:

imc-[nome].py ou imc-[nome].ipynb
